## Структура проекта

- **Lab 1: GCD**: Программа для вычисления наибольшего общего делителя (НОД) двух чисел.

- **Lab 2: HEX-Class**: Класс для работы с шестнадцатеричными числами, включая арифметические операции.

- **Lab 3: Figures-Classes**: Реализация базовых классов для геометрических фигур (треугольник, квадрат, октагон) с использованием наследования и полиморфизма.

- **Lab 4: Figures-Classes-Templates**: Шаблонная версия классов фигур, реализация для пентагона, ромба и трапеции.

- **Lab 5: Forward-List-on-Map**: Реализация односвязного списка на основе std::map с использованием умных указателей.

- **Lab 6: SOLID-design-patterns**: Реализация принципов SOLID с использованием паттернов проектирования для симуляции боев между NPC (bear, heron, desman).

- **Lab 7: SOLID-design-patterns-improved**: Улучшенная версия предыдущей работы с дополнительными паттернами и наблюдателями.

## Требования

- CMake 3.10 или выше
- Компилятор C++ с поддержкой C++17 (например, g++ или clang++)
- Google Test для запуска тестов (автоматически скачивается через CMake)

## Сборка и запуск

Для каждой лабораторной работы выполните следующие шаги:

1. Перейдите в директорию проекта:
   ```
   cd <project_folder>
   ```

2. Создайте директорию для сборки:
   ```
   mkdir build && cd build
   ```

3. Сгенерируйте файлы сборки с помощью CMake:
   ```
   cmake ..
   ```

4. Соберите проект:
   ```
   make
   ```

5. Запустите исполняемый файл:
   ```
   ./<executable_name>
   ```
   (Название исполняемого файла указано в CMakeLists.txt, обычно `LabXX_exe` или подобное)

6. Для запуска тестов:
   ```
   make test
   ```
   или
   ```
   ./tests
   ```
